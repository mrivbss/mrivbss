A modern multi-tenant SaaS platform built for independent barbers and barbershops.

- Public booking system
- Individual barber booking pages
- Appointment management
- Customer management

- Secure authentication
- Row Level Security (RLS)

- Tenant management
- Edge Functions
- Production deployment
